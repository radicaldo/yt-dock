    About yt-dock

    A self-hosted YouTube transcript library — fetch, store, summarize, and search transcripts locally. No API keys, no cloud. View the project on GitHub ↗

    Built with these open-source projects
    • youtube-transcript-api — caption fetching
    • yt-dlp — video metadata
    • sentence-transformers — semantic embeddings
    • sumy — extractive summarization
    • DuckDB — storage + vector similarity
    • FastAPI — the web framework
